# Directioner

Logic Pro에 오디오 파일을 정확한 위치에 삽입하는 macOS 메뉴바 앱.

## 사용법

1. **단축키** (기본: Shift 두 번) 입력
2. 약어 입력 → 파일 선택
3. Logic Pro 타임라인에서 원하는 위치 클릭 → 자동 삽입

## 설치

1. `Directioner.zip` 다운로드 후 압축 해제
2. `Directioner.app`을 `/Applications`로 이동
3. 처음 실행 시 손쉬운 사용(Accessibility) 권한 허용

## 설정

메뉴바 아이콘 → 설정에서 단축키 및 약어 관리.

## 요구사항

- macOS 13 이상
- Logic Pro
